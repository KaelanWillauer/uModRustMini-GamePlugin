# uModRustMini-GamePlugin
Mini-Game system manager developed with the uMod API for Rust.
# Systems
* Spawn system
* Game joining system
* Map voting system
# Gamemodes
* Free For All
* Gun Game
* One In The Chamber
* Infected
* Snipers Only
* Tanks n Nails
